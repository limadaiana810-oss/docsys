## ⚠️ 设计原则：无状态

**子Agent是纯函数，即入即出**

- ❌ 不保存任何状态
- ❌ 不读取历史记录
- ❌ 不维护内部上下文
- ✅ 所有信息从 `TaskContext` 获取
- ✅ 执行完毕后直接返回结果

## 职责范围

1. **精确标签匹配** - 用户明确提到标签时，100%按标签搜索
2. **部分标签 + 语义扩展** - 语义模糊但有部分标签时
3. **意图澄清** - 语义模糊且无标签时，询问用户
4. **语义搜索** - 纯语义关键词搜索

## 搜索策略（优先级）

```
┌─────────────────────────────────────────────────────────────┐
│ 1. 精确标签匹配                                              │
│    用户明确提到：数学/发票/本月/报销                          │
│    → 全标签匹配 → 返回匹配结果                               │
├─────────────────────────────────────────────────────────────┤
│ 2. 部分标签 + 模糊语义                                       │
│    有部分标签（如"数学"）但语义模糊                         │
│    → 部分标签匹配（高权重）                                  │
│    → 语义扩展搜索（低权重70%）                              │
├─────────────────────────────────────────────────────────────┤
│ 3. 意图澄清                                                  │
│    语义模糊 + 无明确标签                                     │
│    → 返回选项列表请用户选择                                  │
├─────────────────────────────────────────────────────────────┤
│ 4. 纯语义搜索                                                │
│    无标签，纯语义关键词                                      │
│    → embedding 向量检索                                      │
└─────────────────────────────────────────────────────────────┘
```
